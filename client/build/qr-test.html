How to Test the QR Code Feature in the App

  1. Go to the Dashboard in the main application
  2. Click on "Preview Email with QR Codes" button
  3. In the preview modal, you'll see QR codes for each venue
  4. Use the "Copy HTML for Email Client" button to copy the HTML
  5. Paste the HTML into an email client to test sending

The QR codes in the email preview are generated using the same API as the ones on this page, ensuring they will be scannable in email clients that support HTML content.
